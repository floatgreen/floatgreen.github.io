Build_my_hugo_site - 2018-04-17 17:00:44

Record the steps of building a hugo site ( 2018-04) get the hugo binary and mv it in some where in PATH # https://github.com/gohugoio/hugo/releases wget https://github.com/gohugoio/hugo/releases/download/v0.39/hugo_0.39_Linux-64bit.tar.gz build a site hugo new Mysite take a look at the directory ▸ archetypes/ ▸ content/ ▸ layouts/ ▸ static/ config.toml make a post hugo new about.md # about.md is generated in content/about.md hugo new post/first.md # first.md is in content/post/first.
