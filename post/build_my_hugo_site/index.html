Build_my_hugo_site

Published 04-17-2018 17:00:44

Record the steps of building a hugo site ( 2018-04)

get the hugo binary and mv it in some where in PATH

# https://github.com/gohugoio/hugo/releases
wget https://github.com/gohugoio/hugo/releases/download/v0.39/hugo_0.39_Linux-64bit.tar.gz

build a site

hugo new Mysite

take a look at the directory

  ▸ archetypes/
  ▸ content/
  ▸ layouts/
  ▸ static/
    config.toml

make a post

hugo new about.md
# about.md is generated in content/about.md
hugo new post/first.md
# first.md is in content/post/first.md

get a theme

cd themes
git clone https://github.com/spf13/hyde.git

local server a site

# in the root directory of Mysite
hugo server --theme=hyde --buildDrafts --watch
# open in browser :http://localhost:1313
# no public/ dir generated at this time

generate pages

hugo --theme=hyde --baseUrl="https://floatgreen.github.io/"
# the default is public/
# can use -d option , e.g. "-d WWW"
# baseURL can also be set in config.toml

push the public/ to github for the first time

cd public
git init
git remote add origin git@github.com:floatgreen/floatgreen.github.io.git
git add -A
git commit -m "first commit"
git push -u origin master

write new posts and update the site (rebuilding site)

#always write new post start from 
hugo new post/new.md

deploy.sh

#! /bin/bash

set -eux 

echo -e "\033[0;32mDeploying updates to GitHub...\033[0m"

msg="rebuilding site `date`"
if [ $# -eq 1  ]
    then msg="$1"
fi

rm -rf public

git clone git@github.com:floatgreen/floatgreen.github.io.git

mv floatgreen.github.io public

#==================
cd public
git rm -r .
cd ..
#==================
# Build the project. 
hugo --theme=hyde --baseUrl="https://floatgreen.github.io/" --buildDrafts

#==================
cd public
git add -A
# Commit changes.
git commit -m "$msg"
git push 
# Come Back
cd ..

add tags, categories in post

# https://discourse.gohugo.io/t/how-to-add-tag-and-category/3202/5

# In your config.toml:

[taxonomies]
   tag = "tags"
   category = "categories"

#Then in the yaml front matter for content/blog/examplepost.md:

title: My Blog Post
categories: [development, publishing]
tags: [hugo,content,static site generator]

highlight the code

  • http://stevenrosenberg.net/hugo/2017_0214_hugo_syntax_highlighting_with_highlight_js/

  • creat a file : layouts/partials/header_includes.html

  • and add the content:

<link rel=“stylesheet” href=“//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.12.0/styles/default.min.css”>
<script src=“//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.12.0/highlight.min.js”></script>
<script>hljs.initHighlightingOnLoad();</script>

add disqus comments

#first get a disqus Shortname in disqus site

#https://floatgreen.disqus.com/admin/settings/general/

#and then set the config.toml:

disqusShortname = "floatgreen"
